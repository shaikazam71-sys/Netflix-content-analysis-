# # 🎬 Netflix Content Analytics Dashboard

## 📌 Overview
This project is an interactive Power BI dashboard built to analyze Netflix content. It provides insights into TV Shows and Movies by visualizing ratings, genres, countries, release years, total views, and average duration. Users can interact with the dashboard using slicers to explore data from different perspectives.

---

## 🎯 Project Objectives
- Analyze Netflix Movies and TV Shows.
- Compare content by Genre and Country.
- Monitor Total Views and Average Ratings.
- Identify content trends across Release Years.
- Build an interactive dashboard using Power BI.

---

## 📊 Dashboard Features
- ✅ KPI Cards
  - Total Movies
  - Total TV Shows
  - Total Views
  - Average Rating
  - Average Duration

- ✅ Interactive Slicers
  - Type (Movie / TV Show)
  - Genre
  - Country

- ✅ Visualizations
  - Count of Ratings by Release Year
  - Genre-wise Content Distribution
  - Country-wise Total Views
  - Movies vs TV Shows Distribution
  - Average Rating Gauge
  - Summary Table (Genre, Type, Views & Rating)
